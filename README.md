# STORE PROGRAM
This app is for stores
It will facilitate the work in the calculations and will give an opportunity to store data about the store
## Installation
```bash
git clone git@github.com:Noro-Hovhannisyan/StoreProgram.git
git cd StoreProgram
python3 main.py
```
